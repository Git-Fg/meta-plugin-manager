# Quick Preset Reference

**Choose the right preset for your task:**

## Development Workflows
- **feature** - Build new functionality with integrated review
- **feature-minimal** - Simple features without ceremony
- **tdd-red-green** - Test-first development
- **spec-driven** - Specifications guide implementation
- **refactor** - Code quality improvement
- **documentation-first** - Docs drive implementation

## Review & Quality
- **review** - Review code changes without modifications
- **security-audit** - Security-focused code review
- **performance-review** - Performance optimization review

## Analysis & Debugging
- **debug** - Investigate and fix specific issues
- **investigation** - Deep codebase understanding
- **tests-improvement** - Enhance test coverage

## Documentation & Knowledge
- **docs** - Create or update documentation
- **readme** - Comprehensive README creation
- **architecture-doc** - Document system architecture

For detailed preset patterns and examples, see [references/detailed-content.md](references/detailed-content.md).
